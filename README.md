## Features

- User Registration & Login
- JWT-based Authentication
- Product Listing
- Product Details Page
- Admin Product Creation
- Form Validation (Express Validator)
- UI (Tailwind CSS)
- RESTful APIs

## Tech Stack

### Frontend

- React (Vite)
- Tailwind CSS
- Redux Toolkit
- Axios
- Swiper

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Express Validator
